# BMI Calculator 🧮

A simple and user-friendly BMI (Body Mass Index) Calculator built using Python and Flask. This web application allows users to enter their height and weight to calculate their BMI and view their weight category instantly.

## Features

- Calculate BMI based on height and weight
- Simple and responsive user interface
- Instant results
- Displays BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Installation

1. Clone the repository:

```bash
git clone https://github.com/vedanthgowda217-sys/BMI-Calculator.git
```

2. Navigate to the project directory:

```bash
cd BMI-Calculator/bmiprojct
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```

## BMI Formula

BMI is calculated using the formula:

```text
BMI = Weight (kg) / Height² (m²)
```
